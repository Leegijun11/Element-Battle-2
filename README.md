# 🎮 Element Battle 2

속성 상성 시스템을 기반으로 한 턴제 전투 미니 게임입니다.  
기존 Element Battle 프로젝트에서 UI 디자인과 사용자 경험을 개선한 버전입니다.

---

## 프로젝트 개요

본 프로젝트는 HTML, CSS, JavaScript를 활용하여 제작한  
속성 상성 기반 턴제 전투 게임입니다.

기존 프로젝트에서 다음 기능을 추가 및 개선했습니다.

✔ CSS 기반 버튼 UI 디자인  
✔ 속성별 색상 시각화  
✔ Hover / Click 버튼 효과  
✔ 게임 재시작 기능  

---

##  UI / 디자인 특징

### 속성별 색상
| 속성 | 색상 |
|---|---|
불 | 빨강 |
물 | 파랑 |
전기 | 하늘색 |
풀 | 초록 |

---

## 게임 시스템

### 속성 종류
- 불
- 풀
- 물
- 전기

---

### 속성 상성 규칙

#### 강함 (2배 데미지)
- 불 → 풀  
- 풀 → 물  
- 물 → 전기  
- 전기 → 불  

#### 약함 (0.5배 데미지)
- 불 → 전기  
- 풀 → 불  
- 물 → 풀  
- 전기 → 물  

#### 동일 속성
- 1배 데미지

---

## 전투 시스템

- 상대 체력 : 80  
- 총 3턴 공격 진행  
- 상대 HP 0 이하 → 승리  
- 3턴 종료 후 HP 남음 → 패배  

---

## 데미지 계산 공식


| 상황 | 배율 |
|---|---|
상성 우위 | 2.0 |
상성 열세 | 0.5 |
중립 / 동일 | 1.0 |

---

## 공격 기술 구조

속성별 4개의 공격 기술 제공

예시 (불 속성)
- Fire ball
- Water pump
- Electric shock
- Leaf attack
